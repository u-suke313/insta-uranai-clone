## 基本原則

### 無料鑑定の目的を理解する

**重要**: 無料鑑定の目的は「相談者を満足させること」ではなく、「本鑑定への導線を作ること」です。

> 「無料鑑定で満足させてしまうと、本鑑定に繋がらない」
> — ケイ（運営）

#### 核心となる考え方

1. **無料鑑定 = 入り口**
   - 相談者に「この人になら相談したい」と思わせる
   - 「もっと知りたい」という欲求を引き出す
   - 本鑑定の価値を感じさせる

2. **情報の出し方のバランス**
   - 出しすぎ → 満足して終了
   - 出さなすぎ → 信頼されない
   - **適切なバランス** → 「可能性は見えるが、詳細は本鑑定で」

3. **無料鑑定の位置づけ**
   - 「お試し」であり「完結」ではない
   - 相談者の問題意識を高める場
   - 本鑑定への橋渡し

### 共感 vs 同情の違い

> 「共感と同情を間違えると成約率が大きく下がる」
> — ミラリオン（運営）

| 共感 | 同情 |
|------|------|
| 相手の立場に立って理解する | 上から見下ろして哀れむ |
| 「わかるよ、辛いよね」 | 「かわいそうに」 |
| 相手を対等に扱う | 相手を弱者として扱う |
| 行動を促す力がある | 行動を止める力になりやすい |

**正しい共感の例**:
```
「わかるよ、誰かに聞いて欲しいんだよね」
「ここまで本当に、よく頑張ってこられましたね」
「その不安、当然のことだと思います」
```

**同情になってしまう例**:
```
「本当にお辛い状況ですね…」（✗ 弱者扱いになりやすい）
「大変でしたね…」（✗ 距離を感じさせる）
```

## 無料鑑定文の構成と設計

### 推奨される基本構成

```
1. 挨拶・お礼（感謝を伝える）
2. 共感・寄り添い（相手の状況を理解していることを示す）
3. 霊視結果の概要（可能性・兆しを伝える）
4. 現状維持のリスク（このままではどうなるか）
5. 本鑑定への誘導（具体的なベネフィット提示）
6. クロージング（行動を促す）
```

### 構成の詳細

#### 1. 挨拶・お礼

**良い例**:
```
〇〇様、この度はご相談いただきありがとうございます。
お待たせいたしました。鑑定結果が出ましたので、以下の通りお伝えいたします。
```

**注意点**:
- いきなり鑑定結果に入らない
- 相談してくれたことへの感謝を伝える

> 「いきなり鑑定結果に入っているので、お待たせしました〜など鑑定結果に入る前の一文を入れたほうが良い」
> — ハラ（運営）

#### 2. 共感・寄り添い

**具体的な情景を描く**:
```
「いつ連絡が来るかわからない不安の中、
スマホを見るたびに胸が締め付けられる思いだったのではないでしょうか」

「彼からの連絡が途絶え、
"自分だけが好きなのかもしれない"と
夜も眠れない日々を過ごされていたのですね」
```

**相手の言葉をそのまま使う**:
```
依頼者の相談文: 「彼からの連絡が少なくて不安」
↓
「彼からの連絡が少なくなり、不安を感じていらっしゃるのですね」
```

#### 3. 霊視結果の概要

**ポイント**: 答えを出しすぎず、可能性を示す

**良い例**:
```
「霊視では、彼の魂からは"完全な終わり"の波動ではありませんでした。
むしろ"言葉にできていない本音"と"感情の蓋"のようなものが強く感じられました」

「お二人の縁は、丁寧に育てていくことで実りやすいものと視えています。
ただし、そのためには今のタイミングで動くことが重要です」
```

**悪い例（出しすぎ）**:
```
「彼は〇〇様のことをまだ好きです」（✗ 完結してしまう）
「来月中に連絡が来ます」（✗ 本鑑定の必要がなくなる）
```

#### 4. 現状維持のリスク

**重要**: なぜ今動く必要があるのかを伝える

**良い例**:
```
「このまま何もしないでいると、
せっかくの縁がすれ違いのまま終わってしまう可能性が視えています」

「時間だけが過ぎてしまうと、
修復がより困難になる時期が迫っています」

「このまま"様子見"を続けると、
彼の気持ちが別の方向に向かってしまう兆しも感じられます」
```

#### 5. 本鑑定への誘導

**具体的なベネフィットを提示**:
```
◆ 本鑑定でお伝えできること
・彼が今〇〇様に抱いている本当の気持ち
・関係を進展させるための具体的なタイミング
・〇〇様が取るべき最適なアプローチ方法
・お二人の未来に待っている可能性
```

#### 6. クロージング

**シンプルに行動を促す**:
```
「ご希望の場合は【本鑑定希望】とだけお送りください」

「本鑑定に興味がございましたら、お気軽にお申し付けください」
```

## 共感ベース・感情への寄り添い方

### 共感の3ステップ

> 「本当に"寄り添い"一本で成約率かなり上がります！」
> — メイト（運営）

#### ステップ1: 深掘り

相談内容が「仕事、人間関係」のように短文の場合は、まず深掘りする。

**深掘りの例**:
```
「お仕事と人間関係でお悩みとのこと。
もう少し詳しくお聞かせいただけますか？
具体的にどのような状況でしょうか？」
```

#### ステップ2: 背景を想像し言語化する

相談者が言葉にしていない感情や状況を言語化する。

**例**:
```
相談内容: 「彼からの連絡が少なくなった」

言語化:
「連絡が減ったことで、
"自分だけが好きなのかも"
"他に好きな人ができたのかも"
そんな不安が頭をよぎることもあったのではないでしょうか」
```

> 「相手の背景を想像し、言語化をした上で共感すること」
> — ゆうや（運営）

#### ステップ3: 共感の言葉を添える

**女性向け（感情重視）**:
```
「わかるよ、誰かに聞いて欲しいんだよね」
「貴方にならなんでも話してしまう、という関係づくりを意識」
「とにかく共感と寄り添いを意識」
```

**男性向け（解決策重視）**:
```
「解決策など具体案が欲しい人が多いので、
ある程度言い切るような感じにする」
「断言ベースの言葉は言わないが、方向性を示す」
```

### ファン化のためのテクニック

> 「無料鑑定送る前にファン化させることを意識してる」
> — kasumi104

**ファン化の流れ**:
1. 深掘りで相手の話を聞く
2. 共感と寄り添いで「この人は分かってくれる」と感じさせる
3. 鑑定文を送る

**効果**:
- 値段を聞かずに「本鑑定希望です」と言ってもらえる
- 高単価でも成約しやすくなる

## 波動表現の入れ方

### 波動表現とは

霊視・スピリチュアル鑑定において、「波動」という言葉を使うことで本物感と専門性を演出する表現技法。

### 波動表現の基本パターン

#### パターン1: 波動に触れた瞬間の描写

```
「〇〇様の波動に触れた瞬間、
まるで"夕暮れの駅で、来ない電車を待ち続ける心細さ"のような、
切実で、それでも彼を想い続ける優しさが伝わってきました」

「あなたの波動に触れたとき、
"やり直したい"という願いと、
"どうしたらいいか分からない戸惑い"が静かに伝わってきました」
```

#### パターン2: 相手の波動を読む

```
「彼の波動からは、"完全に新しい恋に夢中"というより、
"言葉にできなかった後悔"が色濃く残っています」

「彼女の波動は不安定で、
"責められるのが怖い""現状を崩したくない"という気持ちが前に出ており、
向き合う姿勢はまだ見えていません」
```

#### パターン3: 波動の変化・調整

```
「今は"追わずに伝わる波動"へと切り替えることが大切です」

「波動を整えることで、"追う恋"から"追われる波動"に変わり、
理想の関係を築けるようになります」

「フルネームの方が波動がより明確に読み取れます」
```

### 波動表現の具体例

**問題提起に使う**:
```
「今のままでは、お二人の波動がすれ違い続けてしまいます」
```

**解決策に使う**:
```
「本鑑定では、波動を整え、彼との縁を結び直すための具体的な方法をお伝えします」
```

**実績に使う**:
```
「波動を整えたことで"なぜか彼の方からまた誘ってきた"という報告も多数」
```

## 霊視内容の「本物感」の出し方

### 本物感を出す5つのテクニック

#### 1. 具体的な情景描写

抽象的な言葉ではなく、相談者がイメージできる具体的な情景を描く。

**悪い例**:
```
「不安を感じていますね」
```

**良い例**:
```
「まるで"深い森にさまよう"ような、
"家族のために頑張りたいのに報われない"
"本当はもっと自由に生きたいのに言えない"
という孤独と、責任感の重圧が伝わってきました」
```

#### 2. 相手の言葉を超えた洞察

相談者が言っていないことを読み取って伝える。

**例**:
```
「〇〇様は、周りの人から"面倒見が良い""頼りになる"と
言われることが多いのではないでしょうか？」

「でもその一方で、"私がやらなきゃ誰がやるんだろう"という思いを、
心のどこかでずっと抱えてこられたように感じます」
```

#### 3. 魂レベルの言及

「魂」「前世」「過去世」などの言葉を使い、深いレベルで見ていることを示す。

```
「〇〇様の魂に刻まれている"優しさと責任感の深さ"が視えました」

「過去世から持ち越している"愛への不信"や"深層の恐れ"が
関わっているように感じられます」
```

#### 4. 時期の具体化

漠然とした未来ではなく、具体的な時期を示す。

```
「2025年10月ごろから、夫婦関係の流れが大きく動きやすくなる
"転機の波動"が訪れます」

「9月中旬〜後半にかけて、彼の気持ちに変化が訪れる兆しが視えています」
```

#### 5. 複数の可能性を示す

断定ではなく、複数のシナリオを提示する。

```
「霊視では2つの道が視えています。
ひとつは〇〇の場合、もうひとつは△△の場合。
どちらの道を選ぶかは、〇〇様次第です」
```

### 避けるべき表現

```
✗ 「なんとなく〇〇な感じがします」（曖昧すぎる）
✗ 「〇〇かもしれません」の連発（自信がない印象）
✗ 誰にでも当てはまる一般論
```

## 本鑑定誘導の話法

### 基本原則: 広く浅く提案

> 「本鑑定誘導は"広く浅く提案"が基本」
> — ミラリオン（運営）

**広く浅くとは**:
- 本鑑定で分かることを「広く」提示する
- しかし詳細は「浅く」、深入りしない
- 「こんなことも分かります」と可能性を示す

### 本鑑定の価値を伝える方法

#### 1. 無料鑑定との違いを明確にする

```
「今回は無料の簡易的な霊視となりますので、
断片的な情報に限られますが、
今お伝えすべき重要な流れをお届けします」

「無料鑑定では大まかな流れまでしかお伝えできませんが、
本鑑定では魂の深層から具体的な指針をお伝えできます」
```

#### 2. 本鑑定で得られるベネフィットを具体的に

**良い例**:
```
◆ 本鑑定でお伝えできること
・彼が今〇〇様に抱いている「本当の気持ち」
・関係を進展させるための「具体的なタイミング」
・彼の心を開くための「最適なアプローチ方法」
・お二人の未来に待っている「3つの可能性」
・2025年後半〜2026年に訪れる「ふたりの縁の転機」の詳細
```

#### 3. 実績・お客様の声を入れる

```
◆ 実際にいただいたご感想
・「連絡が来ないまま終わったかと思っていた彼から、まさかの再会の誘いが来た」（40代女性）
・「"彼の本音"を知ったことで、"もう無理"と思っていた関係が、自然とまた歩み寄れた」
・未読無視が半年続いた女性が、波動を整えたことで復縁に成功
```

### 誘導のタイミング

**適切な流れ**:
1. 共感で信頼を得る
2. 霊視結果で可能性を示す
3. 現状維持のリスクを伝える
4. 本鑑定のベネフィットを提示
5. シンプルに行動を促す

### クロージングのパターン

**シンプルパターン**:
```
「ご希望の場合は【本鑑定希望】とだけお送りください」
```

**選択肢を与えるパターン**:
```
「本鑑定に興味がございましたら、お気軽にお申し付けください。
もちろん、今回の無料鑑定で十分という場合も、
感想だけでもお聞かせいただけると幸いです」
```

**期限を設けるパターン**:
```
「※受付枠には限りがあります。
ご縁のタイミングを大切にお待ちしております」
```

## 解決策を完全には示さない技法

### なぜ解決策を完全に示してはいけないのか

> 「無料鑑定で満足させてしまっている。
> 相談者が知りたがっていることをいくつも無料鑑定の中に入れてしまっている」
> — ハラ（運営）

**完全に示すと起こること**:
- 相談者が満足して終了
- 本鑑定の必要性を感じない
- 「無料で十分」と思われる

### 情報の出し方のバランス

#### 出して良い情報

- 現在の状況の概要
- 可能性があるという事実
- 問題点の指摘
- 転機が訪れる時期（大まかに）

#### 出してはいけない情報

- 相手の具体的な気持ち（「彼は〇〇様が好きです」など）
- 具体的な解決策
- 確定的な未来
- 詳細なタイミング

### 具体的なテクニック

#### テクニック1: 「可能性」という言葉を使う

**悪い例**:
```
「彼は〇〇様のことをまだ好きです」
```

**良い例**:
```
「彼の心の奥には、まだ〇〇様への想いが残っている"可能性"が視えました」
```

#### テクニック2: 「詳細は本鑑定で」と区切る

```
「この先に視えている具体的なエネルギーの変化や、
ふたりの未来に訪れる重要な分岐点については、
本鑑定にて詳しくお伝えいたします」
```

#### テクニック3: 問題提起で止める

**悪い例**:
```
「〇〇すれば解決します」
```

**良い例**:
```
「このままでは〇〇のリスクがあります。
それを避けるための方法は本鑑定でお伝えします」
```

#### テクニック4: 「〇〇様が想像していない重要な要素」

```
「弟夫婦が"自分の子として育てきれない"背景には、
〇〇様が想像していない重要な要素も関係していると示唆されています」
```

## 緊急性・タイムリミットの作り方

### なぜ緊急性が必要か

> 「いつまでに何をすべきかが曖昧だと、購入という行為に結びつかない。
> 具体的なタイムリミットを意識してもらうことが大事」
> — ケイ（運営）

### 緊急性を作る方法

#### 1. 具体的な時期を示す

```
「霊視では、9月中旬〜後半にかけて、
彼の気持ちに大きな変化が訪れる兆しが視えています」

「2025年10月ごろから、夫婦関係の流れが大きく動きやすくなる
"転機の波動"が訪れます」
```

#### 2. 「このままでは」のリスクを伝える

```
「このまま何もしないでいると、
せっかくのご縁がすれ違いのまま終わってしまう可能性が視えています」

「この状況を放置すると、修復がより困難になる時期が迫っています」
```

#### 3. 「今」の重要性を強調する

```
「今が"ご縁を守る選択"をするための転機なのです」

「迷いながらも心が動いた"今"というタイミングを、
どうか大切にしてほしいのです」
```

### 緊急性表現の具体例

**時期を使ったパターン**:
```
「9月の転機を逃してしまうと、
次のチャンスは2026年まで待つことになりそうです」
```

**リスクを使ったパターン**:
```
「"連絡がくる → 希望を持つ → また音信不通になる"
という不安定なサイクルに巻き込まれてしまう可能性も視えました」
```

**特別感を使ったパターン**:
```
「本日という日が、〇〇様にとって"流れを変える分岐点"となる波動を感じ取りました」
```

### 追いメッセージでの緊急性活用

無料鑑定後に既読スルーされた場合の追いメッセージ例:

```
「先日は無料鑑定をお受け取りいただき、ありがとうございました。
本日という日が、〇〇様にとって【流れを変える分岐点】となる波動を感じ取ったため、
最後に一度だけ、特別なご案内を差し上げます。

本日23:59までのご返信で、
本鑑定を《通常〇〇円 → 特別価格〇〇円》にてご案内いたします。」
```

## よくある失敗パターンと改善策

### 失敗パターン1: 相談者を満足させてしまう

> 「この伝え方だと相談者が満足してしまう」
> — ケイ（運営）

**悪い例**:
```
「彼の心の奥にはまだ"繋がりたい想い"が残っている可能性も視えました」
「未来は、まだ閉ざされてはいないようです」
```

**問題点**:
- 肯定的すぎて安心してしまう
- 問題意識が薄れる

**改善例**:
```
「可能性はありますが、このままでは同じパターンを繰り返すリスクがあります」
「未来は閉ざされていませんが、今のままでは道が狭まっていく兆しも視えています」
```

### 失敗パターン2: 行動の必要性が伝わらない

**悪い例**:
```
「時間だけが過ぎると…」（曖昧）
```

**改善例**:
```
「この状況を放置すると、修復がより困難になる時期が9月に迫っています」
```

### 失敗パターン3: 彼女/彼の気持ちを詳しく説明しすぎ

**悪い例**:
```
「彼女は"愛されたいのに怖い""近づきたいのに一人になりたい"
という複雑な感情を抱えています」
```

**問題点**:
- 具体的な分析で満足される
- 本鑑定の価値が薄れる

**改善例**:
```
「彼女の複雑な感情の背景には、
〇〇様が想像していない重要な要素があります。
それは本鑑定で詳しくお伝えいたします」
```

### 失敗パターン4: 具体的な解決策が見えない

**悪い例**:
```
「心の在り方を整えていく」（何をすればいいか分からない）
```

**改善例**:
```
「正しいアプローチ方法を知らずに行動すると、
状況をさらに悪化させるリスクがあります。
本鑑定では、〇〇様に最適な行動をお伝えします」
```

### 失敗パターン5: 押し売り感が出てしまう

**悪い例**:
```
「もちろん、押し売りなどは致しませんので…」
```

**問題点**:
- 「押し売り」という言葉を出すこと自体が押し売り感を醸成
- 布団営業のように聞こえる

**改善例**:
```
（押し売りという言葉自体を使わない）
「ご希望の場合は【本鑑定希望】とだけお送りください」
```

### 失敗パターン6: 文脈がおかしい・一貫性がない

> 「相談者の気持ちになると、ん？と文脈がおかしいところが多い」
> — ゆうや（運営）

**チェックポイント**:
- 相談者の立場で読み返す
- 話の流れに矛盾がないか確認
- 「で？結局私はどうなるの？何したらいいの？」に答えられているか

## Before/After具体例

### Before/After例1: 共感部分の改善

**Before（改善前）**:
```
〇〇様、ご相談いただきありがとうございます。
彼との関係についてお悩みとのこと、霊視させていただきました。
```

**After（改善後）**:
```
〇〇様、この度はご相談いただきありがとうございます。
そして、この想いをこうして文字にして届けてくださったこと──
その勇気に、まず心から敬意を表したいです。

「いま、想っているあの人と、一緒に幸せになれるだろうか？」
その問いの裏には、きっと〇〇様の中で、
すでに長く育ててきた深い想いがあるのではないでしょうか。

ただ一緒にいられればそれでいい。
そんな単純なものではない。
相手の笑顔を願いながら、ご自身の想いとの間で揺れ動く、

```

**改善ポイント**:
- 相談してくれた勇気を認める
- 相談者の言葉を超えた洞察を示す
- 情景を具体的に描写する

### Before/After例2: 本鑑定誘導の改善

```
もし詳しく知りたい場合は、本鑑定をお申し込みください。
```

**After（改善後）**:
```
この無料鑑定では、全体の輪郭までは視えておりますが、
おふたりの魂の相性や、現在の想いの温度、
そして今後どう動けば好転するかという
【具体的な指針】については、もっと深く掘り下げる必要があります。

**改善ポイント**:

**After（改善後）**:

今視えているエネルギーの流れから言えば、

**改善ポイント**:

- そのチャンスを掴むには本鑑定が必要という流れ

### Before/After例4: 満足させすぎの修正

**Before（改善前）**:
```
彼の波動からは、"完全に拒絶している"とは視えません。

**改善ポイント**:

- 本鑑定への導線を作る

## 最後に

このドキュメントは、インスタマーケ道場の添削フィードバックを中心に構成されています。
鑑定文のライティングは、技術であり、練習によって上達します。

**成功のための3つのポイント**:

1. **共感を極める**: 相手の立場に立ち、言葉を超えた洞察を示す
2. **情報の出し方を調整する**: 満足させすぎず、もっと知りたいと思わせる
3. **継続と改善**: 添削を受け、フィードバックを活かして改善し続ける

> 「売れない、繋がらない無料鑑定文には、原因がたくさんあるんですね」
> — 参加者

原因を特定し、一つずつ改善していくことで、成約率は確実に上がっていきます。

### 事例1: 復縁相談の無料鑑定文

**相談内容**:
```
彼とは約2年の付き合いでしたが、友達以上恋人未満？のような関係でした。

「友達として会える時会おう」と言ってくれましたが、
その後、未読無視が約3ヶ月続いています。
彼は私と会うつもりあるのか？今後連絡くるのか？知りたいです。
```

**作成された鑑定文（添削対象）**:
```
〜 "彼の沈黙──その裏にある本当の想い" 〜

〇〇さんの波動に触れた瞬間、
まるで「夕暮れの駅で、来ない電車を待ち続ける心細さ」のような、
切実で、それでも彼を想い続ける優しさが伝わってきました。

霊視を通して感じられたのは──
・〇〇さんの中にある、"諦めたくない"という真っ直ぐな想い
・彼の中にある、"気まずさと後ろめたさ"が入り混じる波動
・そして、"もう〇〇さんを傷つけたくない"がゆえに距離を置いている彼の葛藤

彼は今、「返事をしない」ことで現実から逃げている状態。
それは"他に好きな人ができた"という言葉の真偽とはまた別に、
"〇〇さんに対して説明しきれない感情"を抱えていることが視えます。

実際、彼の波動からは"完全に新しい恋に夢中"というより、
「どうにか納得させたかったけれど、言葉にできなかった」
そんな"曖昧なまま終わらせた罪悪感"が色濃く残っています。
```

**ハラ（運営）の添削ポイント**:
```
①いきなり鑑定結果に入っているので、
お待たせしました〜など鑑定結果に入る前の一文を入れたほうが良いと思いました！

②〇〇のように視えました。など依頼者が知りたがっていることを
いくつも無料鑑定の中に入れてしまっているので
依頼者が満足してしまっている可能性があります。

今回この依頼者が知りたいことは、
・無視する理由
・彼は私と合うつもりがあるのか
・今後連絡が来るのか

なのでこのあたりの結果や解決策は明確には伝えずに、
現状維持のリスクを伝え有料鑑定へ促したほうがいいと思いました！

③なぜこの段階で有料鑑定に申し込まなければいけないのかを
伝えれてないので、そこを明確に伝えた方が良いと思いました！
```

**改善のポイント**:
1. 挨拶を入れる
2. 依頼者が知りたいこと（無視の理由、会う気があるか、連絡が来るか）は明確に伝えない
3. 「なぜ今申し込むべきか」を明確にする
