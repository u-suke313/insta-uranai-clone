## 無料鑑定の集客施策

### 営業DMの基本戦略

#### 基本的な考え方

営業DMは、占いビジネスにおける最も効果的な集客手段の一つです。メイトによると、「待っていても顧客は来ない」という前提に立ち、積極的にアウトバウンドでアプローチすることが重要です。

#### ターゲットの選定方法

**占い系リールへのコメント者をターゲットにする方法**

1. 占い系の大手アカウントのリールを探す
2. そのリールにコメントしている人をリストアップ
3. 「無料鑑定やってます」とDMを送信

この方法のメリット：
- すでに占いに興味があることが確認できる
- コメントするアクティブユーザーである
- 競合との差別化が図れる

**具体的なDM文面例**

```
初めまして！
占い師の〇〇と申します。

投稿を拝見してDMさせていただきました。
現在、無料鑑定のモニターを募集しております。

もしよろしければ、お悩みをお聞かせいただけませんか？
丁寧に鑑定させていただきます。

ご興味ございましたら、お気軽にご返信ください！
```

#### DMの送信タイミング

- 朝の時間帯（7:00〜9:00）
- 昼休み（12:00〜13:00）
- 夜の時間帯（20:00〜23:00）

特に夜の時間帯は反応率が高い傾向にあります。

#### 営業DMの注意点

1. **スパム判定を避ける**
   - 同じ文面を大量送信しない
   - 文面にバリエーションを持たせる
   - 送信間隔を空ける（最低でも30秒〜1分）

2. **ブロック対策**
   - 無理に売り込まない
   - 興味がない場合は深追いしない
   - 丁寧な言葉遣いを心がける

3. **効果測定**
   - 送信数、返信率、成約率を記録
   - 文面のA/Bテストを実施
   - 時間帯別の効果を分析

#### メイトの営業DM成功事例

メイトは以下の戦略で高い反応率を実現しています：

1. プロフィールを充実させてから営業
2. いくつかの投稿を事前にいいね
3. ストーリーズにリアクション後にDM
4. パーソナライズした文面の使用

### いいね狙い投稿の活用

#### 基本原理

Instagramのアルゴリズムは、エンゲージメント（いいね、コメント、保存）が高い投稿を優先的に表示します。いいね狙い投稿を活用することで、アカウント全体の露出を増やし、無料鑑定募集ポストの表示回数を増加させることができます。

#### いいね狙い投稿の種類

**1. 共感系投稿**

```
「分かりすぎる」と思ったらいいね！

・朝起きるのがつらい
・仕事行きたくない
・でも頑張らなきゃ

あなたは毎日頑張ってます。
自分を褒めてあげてください。
```

**2. 診断系投稿**

```
あなたの誕生月で分かる「2024年の運勢」

1月生まれ：〇〇
2月生まれ：〇〇
...

該当したらいいね！
```

**3. あるある系投稿**

```
占い好きあるある

・気になる人ができると即相性占い
・朝の星座占いは欠かさない
・「当たってる！」が口癖

共感したらいいね！
```

**4. 選択系投稿**

```
直感で選んで！

A. 🌙 月
B. ⭐ 星
C. ☀️ 太陽

選んだものであなたの性格が分かります！
コメントで教えてください。
```

#### いいね狙い投稿の効果

1. **アルゴリズムへの好影響**
   - フォロワーのフィードに優先表示
   - 発見タブへの露出増加
   - 全体的なリーチ向上

2. **フォロワーとの関係構築**
   - 双方向のコミュニケーション
   - 親近感の醸成
   - 信頼関係の構築

3. **無料鑑定募集への誘導**
   - いいね狙い投稿でエンゲージメント向上
   - 次の投稿（無料鑑定募集）がより多くの人に表示
   - 結果として無料鑑定の申し込み増加

#### 投稿頻度とバランス

メイトが推奨する投稿バランス：

- いいね狙い投稿：週3〜4回
- 価値提供投稿（占い解説など）：週2〜3回
- 無料鑑定募集：週1〜2回
- 成果・感想紹介：週1回

#### 効果的な投稿時間

- 朝：7:00〜8:00
- 昼：12:00〜13:00
- 夜：20:00〜22:00

特に夜の時間帯がエンゲージメント率が高い傾向にあります。

### ストーリーズ活用術

#### ストーリーズの重要性

ストーリーズは24時間で消えるため、フォロワーが見逃したくないという心理が働きます。また、フィード投稿よりもカジュアルなコミュニケーションが可能です。

#### 効果的なストーリーズの種類

**1. 質問ボックス**

```
「今のお悩み教えてください」

質問ボックスでお悩みを募集し、
いくつかをストーリーズで回答。
→ 親近感と専門性をアピール
```

**2. 投票機能**

```
「今日の運勢、気になる？」
・気になる！
・毎日チェックしてる！

投票結果に基づいて次のストーリーズで運勢を投稿
```

**3. カウントダウン**

```
「無料鑑定募集まであと〇時間！」

カウントダウン機能を使って
募集開始を告知。
→ 期待感を醸成
```

**4. 感想紹介**

```
購入者からの感想スクリーンショットを
ストーリーズで紹介。

「嬉しいお声をいただきました！」
→ 社会的証明の活用
```

#### ハイライトの活用

ストーリーズは24時間で消えますが、ハイライトに保存することで永続的に表示できます。

**推奨ハイライト構成**

1. **自己紹介**
   - 占い師としての経歴
   - 得意な占術
   - 鑑定への想い

2. **鑑定の流れ**
   - 申し込み方法
   - 鑑定の進め方
   - 納期について

3. **お客様の声**
   - 感想のスクリーンショット
   - ビフォーアフター
   - リピート率の紹介

4. **よくある質問**
   - 料金について
   - 鑑定内容について
   - キャンセルポリシー

5. **無料鑑定について**
   - 現在募集中かどうか
   - 申し込み方法
   - 鑑定内容の説明

### リール戦略

#### リールの重要性

リールはInstagramで最も拡散力のあるコンテンツ形式です。フォロワー以外にもリーチできるため、新規顧客獲得に効果的です。

#### 占いビジネスで効果的なリール

**1. 占い解説系**

```
「手相の見方、知ってますか？」

・生命線の意味
・感情線の見方
・運命線の解釈

15秒〜30秒でわかりやすく解説
```

**2. ビフォーアフター系**

```
「鑑定を受けてこう変わりました」

Before: 彼との関係に悩んでいた
After: 自分に自信が持てるように

※プライバシーに配慮した表現で
```

**3. 今日の運勢系**

```
「今日の〇〇座の運勢」

毎朝投稿することで
ルーティン化させる
```

**4. あるある系**

```
「占い師あるある」

・当たりすぎて自分でも怖い
・友達から無料鑑定を頼まれがち
・星座を聞くのが癖になってる

共感を誘いいいね・保存を促進
```

#### リールからの導線設計

1. リールで興味を持たせる
2. プロフィールに誘導
3. ハイライトで詳細確認
4. DMで無料鑑定申し込み

## ヒアリングから鑑定結果の自動化

### iメイトの基本設定

#### iメイトとは

iメイトは、InstagramのDM対応を自動化できるツールです。特定のキーワードやアクションに対して、自動でメッセージを送信することができます。

#### 基本的な自動化フロー

```
1. 無料鑑定希望のDM受信
   ↓
2. 自動でヒアリングテキスト送信
   ↓
3. 回答を受信
   ↓
4. 鑑定結果を送信（手動 or テンプレート）
   ↓
5. 有料鑑定の案内を送信
```

#### 初期設定の手順

**ステップ1：トリガーの設定**

- 特定のキーワード（「無料鑑定」「鑑定希望」など）
- ストーリーズへのリアクション
- 投稿へのコメント

**ステップ2：自動返信メッセージの設定**

```
お問い合わせありがとうございます！
無料鑑定のご希望ですね。

以下の質問にお答えください。

①お名前（ニックネーム可）
②生年月日
③現在のお悩み

ご回答いただき次第、鑑定を開始いたします。
```

**ステップ3：遅延設定**

即レスではなく、10分後に自動返信する設定がおすすめです（詳細は「待たせる効果」セクションで解説）。

### 人間味を残す工夫

#### 自動化のデメリット

自動化は効率的ですが、「機械的」「冷たい」という印象を与えるリスクがあります。特に占いという個人的なサービスでは、人間味が重要です。

#### 人間味を残すテクニック

**1. 遅延返信の設定**

即レスは「自動だな」とバレやすい。10分後に返信する設定にすることで、「ちゃんと確認してくれている」感を演出できます。

**2. 絵文字・顔文字の適度な使用**

```
NG例：お問い合わせありがとうございます。
OK例：お問い合わせありがとうございます😊
```

**3. 名前の呼びかけ**

```
〇〇様、ご回答ありがとうございます！
```

**4. 個別のコメント追加**

テンプレートの最後に、個別のコメントを一言追加：

```
PS. 〇〇様の強さを感じました。きっと大丈夫です。
```

**5. 音声メッセージの活用**

テキストだけでなく、音声メッセージを送ることで人間味がアップ：

```
「〇〇さん、鑑定結果をお送りしますね。
 少し長くなりますが、大切なことをお伝えしたいので
 最後まで読んでいただけると嬉しいです。」
```

## 成約率向上の戦略

### 10%から30%への改善事例

#### 事例：amato_31さんの成功

amato_31さんは、成約率10%から25〜30%への改善を実現しました。

**改善前の状況**
- 無料鑑定後にそのまま有料案内
- 価格は一つのみ提示
- 成約率：約10%

**改善後の施策**
- 無料鑑定の質を向上
- 松竹梅の価格提示
- 待たせる効果の活用
- ハイライトでの感想紹介
- 成約率：25〜30%

#### 成約率向上の基本原則

1. **無料鑑定の満足度を上げる**
   - 手抜きしない
   - 具体的なアドバイスを含める
   - 「もっと知りたい」と思わせる

2. **信頼関係を構築する**
   - 丁寧なコミュニケーション
   - 相手の話を聞く姿勢
   - 専門性のアピール

3. **適切な提案をする**
   - 押し売りしない
   - ニーズに合った提案
   - 複数の選択肢を提示

---

### 成約率を上げる具体的テクニック

#### テクニック1：無料鑑定の質を上げる

無料だからといって手を抜かない。むしろ、無料鑑定こそ全力で取り組むことが重要です。

**質の高い無料鑑定の特徴**

```
・具体的な分析
・実践的なアドバイス
・共感と理解
・今後の展望
```

**NG例：質の低い無料鑑定**

```
「〇月生まれのあなたは真面目な性格です。
 恋愛運は普通です。頑張ってください。」

→ 一般的すぎて価値を感じない
```

**OK例：質の高い無料鑑定**

```
「〇〇様の星回りを拝見しました。

 現在、火星が恋愛を司る第5ハウスに位置しており、
 積極的なアプローチが実を結びやすい時期です。

 特に〇月〇日〜〇月〇日は、
 新しい出会いのチャンスが高まります。

 ただし、水星逆行の影響で、
 コミュニケーションのすれ違いに注意が必要です。

 具体的には...」

→ 具体的で、行動につながるアドバイス
```

#### テクニック2：段階的な価値提供

いきなり有料を勧めるのではなく、段階的に価値を提供します。

```
ステップ1：無料鑑定（基本的な分析）
    ↓
ステップ2：追加の無料情報（ワンポイントアドバイス）
    ↓
ステップ3：有料鑑定の提案（より詳細な分析）
```

#### テクニック3：ハイライトで感想を見せる

有料鑑定の案内時に、ハイライトに誘導します。

```
「有料鑑定についてはこちらのハイライトをご覧ください。
 過去のお客様の感想も載せていますので、
 ご参考になれば幸いです。」
```

**ポイント：待ち時間の活用**

鑑定結果を待っている間に、ハイライトを見てもらうことで、購入意欲を高めることができます。

```
「鑑定には少しお時間をいただきます。
 お待ちいただいている間に、
 ハイライトの『お客様の声』をご覧いただけると嬉しいです。」
```

#### テクニック4：適切なタイミングでの提案

無料鑑定の結果を送った直後ではなく、少し間を置いてから有料を提案します。

```
タイミング1：鑑定結果送信
    ↓（数時間〜1日）
タイミング2：フォローメッセージ
    「鑑定結果、いかがでしたか？」
    ↓（相手の反応を待つ）
タイミング3：有料鑑定の案内
    「もしよろしければ、詳細鑑定もご用意しております」
```

#### テクニック5：購入の障壁を下げる

```
・分割払いの対応
・お試し価格の設定
・返金保証（条件付き）
```

### 失敗パターンと改善策

#### 失敗パターン1：押し売り

**症状**
```
「有料鑑定もありますよ！」
「今なら割引です！」
「絶対おすすめです！」
```

**問題点**
- 信頼を損なう
- ブロックされるリスク
- 悪い口コミにつながる

**改善策**
```
・相手のニーズを確認してから提案
・選択肢を提示するスタンス
・「ご興味があれば」という言い方
```

#### 失敗パターン2：無料鑑定の手抜き

**症状**
```
・一般的すぎる内容
・コピペ感が強い
・具体性がない
```

**問題点**
- 価値を感じてもらえない
- 有料への期待が持てない
- 二度と依頼されない

**改善策**
```
・個別の情報を必ず入れる
・具体的なアドバイスを含める
・時間をかけて丁寧に作成
```

#### 失敗パターン3：価格提示のミス

**症状**
```
・いきなり高額を提示
・価格帯が一つだけ
・価格の根拠がない
```

**問題点**
- 価格で引かれる
- 選択肢がないと感じる
- 値段に見合う価値が伝わらない

**改善策**
```
・松竹梅で複数の選択肢を提示
・各価格帯の違いを明確に
・価値から価格を説明
```

## 待たせる効果の活用

### 心理的効果の原理

#### 待つことで価値が上がる

人間心理として、「すぐに手に入るもの」よりも「待たなければ手に入らないもの」に価値を感じる傾向があります。

**例**
- 予約の取れないレストラン
- 行列のできるラーメン店
- 限定品・予約販売

占いビジネスでも、適度に「待たせる」ことで、鑑定結果への期待値と価値を高めることができます。

#### 即レスのデメリット

1. **自動化がバレる**
   - 「botだな」と思われる
   - 人間味がなくなる

2. **価値が下がる**
   - 「すぐできるなら大したことない」
   - 希少性が感じられない

3. **機会損失**
   - 待ち時間を活用できない
   - ハイライト等への誘導ができない

## 返報性の原理の活用

### 無料提供の設計

#### 返報性の原理とは

人間には、「何かをもらったらお返しをしなければならない」という心理が働きます。これを返報性の原理と言います。

**例**
- スーパーの試食 → 購入してしまう
- 無料サンプル → 製品を買いたくなる
- 親切にされる → お返しをしたくなる

占いビジネスでは、無料鑑定で価値を提供することで、有料鑑定を購入してもらいやすくなります。

#### 効果的な無料提供

**ポイント1：本当に価値のあるものを提供**

```
NG：形だけの無料鑑定
    「〇月生まれのあなたは〇〇な性格です」

OK：価値のある無料鑑定
    「あなたの星回りから見ると、
     〇月は転機の時期です。
     特に〇日〜〇日は重要な決断を
     迫られる可能性があります。」
```

**ポイント2：期待以上の提供**

```
「無料鑑定なのにこんなに詳しく！」
と思ってもらえるレベルを目指す
```

**ポイント3：惜しみなく提供**

```
「これ以上は有料で」という出し惜しみは逆効果。
無料の範囲内で最大限の価値を提供する。
```

### 価値提供と成約の関係

#### 価値提供の量と成約率の関係

メイトの経験によると、価値提供の量と成約率には相関関係があります。

```
価値提供が少ない → 成約率が低い
価値提供が多い → 成約率が高い
```

**ただし注意点**

価値提供が多すぎると：
- 無料で満足してしまう
- 有料の必要性を感じない

適切なバランスが重要です。

#### 適切な価値提供のバランス

**無料鑑定で提供すべき内容**

```
✓ 具体的な分析
✓ 実践的なアドバイス
✓ 今後の見通し（概要）
```

**有料鑑定で提供すべき内容**

```
✓ より詳細な分析
✓ 複数項目の鑑定
✓ 個別のフォロー
✓ 質問への回答
```

**無料と有料の境界線**

```
無料：「何をすべきか」のヒント
有料：「どうすべきか」の詳細
```

## よくある質問と回答

### Q1：無料鑑定で満足されて有料が売れない

**A**：無料鑑定の内容を見直してください。「もっと知りたい」と思わせる構成になっていますか？

具体的には：
- 概要は伝えるが詳細は有料で
- 「実は他にも気になる点があります」と示唆
- 「より詳しく知りたい方へ」と自然な誘導

### Q2：価格が高いと言われる

**A**：価格を下げるのではなく、価値を上げましょう。

具体的には：
- 鑑定内容を充実させる
- 特典を追加する
- 感想を見せて価値を証明

それでも言われる場合は、松竹梅の「竹」プランを提案。

### Q3：営業DMを送るとブロックされる

**A**：文面とターゲットを見直しましょう。

チェックポイント：
- 売り込み感が強すぎないか
- 相手にとってのメリットが伝わっているか
- ターゲットが適切か（占いに興味がある人か）

### Q4：成約率が10%以下で伸びない

**A**：以下のポイントを確認してください。

1. 無料鑑定の質は十分か
2. 価格提示のタイミングは適切か
3. 松竹梅で提示しているか
4. 待たせる効果を活用しているか
5. ハイライトに感想を掲載しているか

### Q5：リピーターが増えない

**A**：フォローアップを強化しましょう。

具体的には：
- 購入者リストを作成
- 定期的にフォローメッセージを送る
- 季節や誕生日に合わせてアプローチ
- リピーター限定の特典を用意

## K. セールス前の6種類の顧客教育

> **出典**: note記事 `note_37286`

成約率を高めるためには、セールスの前に顧客を「教育」することが重要です。以下の6つの教育を意識しましょう。

### K.1 信用の教育

**目的**: 「この人は信頼できる」と思ってもらう

**方法**
- 実績を見せる（お客様の声、鑑定件数）
- 自己開示（なぜ占いを始めたか、過去の経験）
- 一貫性のある発信

### K.2 目的の教育

**目的**: 「何のためにこのサービスがあるか」を理解してもらう

**方法**
- 占いの本当の価値を伝える（未来を知るだけでなく、行動指針を得る）
- 「なりたい自分になるための手段」として位置づけ

### K.3 問題の教育

**目的**: 顧客自身が「今のままではダメだ」と気づく

**方法**
- 現状の問題点を言語化する投稿
- 「このまま放置すると...」という危機感の共有
- 同じ悩みを持つ人の事例紹介

### K.4 手段の教育

**目的**: 「占いで解決できる」と理解してもらう

**方法**
- 占いで解決した具体例を紹介
- ビフォーアフターを見せる
- 他の手段との違いを明確に

### K.5 投資の教育

**目的**: 「お金を払う価値がある」と納得してもらう

**方法**
- 価格以上の価値があることを伝える
- 「投資」という視点を持たせる
- 無料で得られるものとの違いを明確に

### K.6 行動の教育

**目的**: 「今すぐ行動しなければ」と思ってもらう

**方法**
- 限定性を伝える（期間限定、人数限定）
- 行動しないリスクを伝える
- 背中を押すメッセージ

### K.7 6つの教育の活用例

**投稿順序の例**
1. 月曜：信用の教育（お客様の声紹介）
2. 火曜：問題の教育（悩みに共感する投稿）
3. 水曜：目的の教育（占いで得られる未来）
4. 木曜：手段の教育（解決事例の紹介）
5. 金曜：投資の教育（価値を伝える）
6. 土曜：行動の教育（限定オファー告知）

占いビジネスにおける成約化・セールス戦略の要点は以下の通りです：

### 集客

3. **待たせる効果**で価値を高める

1. **ダウンセル**で取りこぼしを防ぐ
